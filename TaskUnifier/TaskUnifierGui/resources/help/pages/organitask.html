Synchronize with OrganiTask

TaskUnifier now synchronizes with OrganiTask.
OrganiTask is a new website that helps you manage your tasks and improve your productivity.

You can import your tasks from Toodledo into OrganiTask in just 1 click.
OrganiTask uses the latest technologies and offers an amazing user-interface.

Visit OrganiTask's website now: http://www.organitask.com

If you want to use OrganiTask with TaskUnifier:

  • Create an OrganiTask account (and import your Toodledo tasks in 1 click)
  • In TaskUnifier, go to File -> Synchronize with... -> OrganiTask
  • That's all, you can now use TaskUnifier with OrganiTask
