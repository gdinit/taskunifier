Configuration

A start date event is an event representing the start date of a task.
A due date event is an event representing the due date of a task. Please check this section for more information.

If you use the Google Calendar plugin on multiple computers with the same Google account, it is highly recommended to use the same settings !

Email:
Email of your Google Calendar account.
Calendar name:
The plugin will create or edit the calendar with this name.
Create a day event:
If you do not use start time nor due time and if the task start date and due date is the same day, a day event will be created.
Publish start date events:
As explained in this section, tasks will be split into two types of events. If this option is checked, start date events will be published.
Publish end date events:
As explained in this section, tasks will be split into two types of events. If this option is checked, end date events will be published.
Do not publish events older than (days):
Events with a date older than this number of days won't be published.
Do not publish events farther than (days):
Events with a date farther than this number of days won't be published.
Block time in the calendar:
Defines whether the events have to block the time or not.
Start date reminder method:
Defines the method to use for the start date reminders.
Due date reminder method:
Defines the method to use for the due date reminders.
Start event color:
Defines the color of the start date events.
Due event color:
Defines the color of the due date events.
