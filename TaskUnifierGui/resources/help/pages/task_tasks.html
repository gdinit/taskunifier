Task Linked Tasks

The task linked tasks allow you to link tasks to tasks

There are four columns in this table.
The link column contains the description of the link between the "linked task" and the task.
The task column contains the linked task.
If you double click on the select column, the task will be selected.
If you double click on the edit column, the task edit window will be opened.

You can drag and drop tasks from the task table directly in the view:
First, lock (lock button) this view with the selected task, then select tasks in the task table and drag and drop them into this view.
When you have finished, unlock the view (lock button).
