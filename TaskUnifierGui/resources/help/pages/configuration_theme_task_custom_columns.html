Theme - Task custom columns

This is a pro feature.
It allows you to add and remove custom fields to your tasks.
For example, you could add a column "Supervisor" containing a contact person.

There are many type of field you can add:

  • Calendar (date and time)
  • Calendar (date only)
  • Checkbox
  • Contact
  • Context
  • Contexts
  • File
  • Folder
  • Goal
  • Goals
  • Integer
  • Location
  • Locations
  • Note
  • Number of minutes
  • Percentage
  • Priority
  • Real number
  • Star checkbox
  • Tags
  • Task
  • Textfield
  • Time
  • Timer

When you have added your custom fields, restart TaskUnifier and the new fields will be displayed in the task table and in the task edit window.
