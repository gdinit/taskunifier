Date

Timezone:
Your timezone.
Date format:
Date format used to display dates.
Time format:
Time format used to display hours.
Show day of week in dates:
All the displayed dates in TaskUnifier will contain the day of the week.
Use due time:
If this option is not checked, the due dates won't contain any time information.
Ex: 23/01/2011
Use start time:
If this option is not checked, the start dates won't contain any time information.
Ex: 23/01/2011
Day start hour:
Business day start hour.
Day break hour:
Business day break hour (defines the half of the day).
Day end hour:
Business day end hour.
Display a reminder when start/due date is reached event if "No Reminder" is set:
If this option is checked, when the start or the due date is reached, the task reminder window will be opened even if "No Reminder" is set.
In this case, "No Reminder" actually means "0 Minute" reminder.
Display the overdue tasks in the reminder list:
By default, overdue tasks won't be displayed in the reminder list.
If this option is checked, the overdue tasks will be displayed.
