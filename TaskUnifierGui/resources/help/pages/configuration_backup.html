Backup

Auto save every (in minutes):
Defines the duration between each automatic save of the data.
Create a new backup:
Create a new backup that you will be able to restore later if needed.
Backup before each sync:
Create a new backup before each synchronization.
Auto backup every (in hours):
If the last backup is more than X hours old, a new backup will be automatically created.
Number of backups to keep:
Maximum number of backups that will be kept, the oldest ones will be removed.
Backup list:
List of backups that can be restored.
Restore from selected backup:
Restore the selected backup in the above list.
