Task Note

In order to edit a note, click on the edit icon. Then you will be able to edit the note and to add styles to your note. Styles:

Bold
<b>Your text in bold</b>
Italic
<i>Your text in italic</i>
Ordered list
<ol><li>First element</li><li>Second element</li></ol>
Unordered list
<ul><li>First element</li><li>Second element</li></ul>
