Task Searcher

Icon:
You can select an image file on your computer.
The icon will be displayed in the list next to the title.
Title:
The title of you search.
Folder:
The folder (different from task/note folders) where to store the searcher.
You can define sub-folders with a slash (/) seperator between the folders.
Example: A/B/C
The searcher will be stored inside the folder A, sub-folder B, sub-sub-folder C.
Template:
You can specify a template for a searcher, if you drag and drop tasks onto the searcher, the selected template will be applied.
Sort order:
The sort order table allows you to order the tasks.
Order: the order of the column sorts
Column: the column you want to sort
Sort order: ascending or descending sort
Filters:
The filters tree allows you to filter the tasks.
The filters are composed of sub-filters (branches) and conditions.

You can right click on a branch to choose between AND or OR.
You can click on a condition and edit the values at the bottom of the screen.
Filter column:
When you click on a condition in the filter tree, you can edit this field. This field defines on which column the condition will be executed.
Filter condition:
When you click on a condition in the filter tree, you can edit this field. This field defines what type of condition will be executed.
Filter value:
When you click on a condition in the filter tree, you can edit this field. This field defines the value that will be checked.
Depending on the selected column, you will be able to choose one of the values in the list or to enter a value manually.

These are the columns that require a "manual" value:
Title, tags, repeat, note
Anything you want...
Completed on, start date, due date...
You have to enter a number (of days/weeks/months).
The date used in the condition will be : now + number of days/weeks/months.
Note: if you enter nothing, it always means "EQUALS NO DATE"
Reminder, length, importance
You have to enter a number (of minutes for the reminders and the length).
Progress
You have to enter a number between 0 and 1 (eg: 0.54 means 54%).
Contacts
You can enter the name of a contact or the link.
You can also search on a specific contact with a specific link: "name (link)".
Examples:
Contacts contains "Nick"
Includes all the tasks with at least "Nick" in one of the contacts
Contacts contains "Delegated"
Includes all the tasks with at least "Delegated" as one of the links
Contacts contains "Nick (Delegated)"
Includes all the tasks containing the contact "Nick" with the link "Delegated"
