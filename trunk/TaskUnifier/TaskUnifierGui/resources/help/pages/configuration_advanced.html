Advanced

Enable Growl (Mac OS and Windows only):
Enable Growl notifications or not.
Enable Snarl (Windows only):
Enable Snarl notifications or not.
Show reminders:
If unchecked, the task reminder window will never be automatically opened.
Enable communicator:
Enable the communication between TaskUnifier and other softwares.
This is required for all the addons (http://www.taskunifier.com/index.php?page=addons).
Communicator port:
Port to communicate with other softwares (if you change this, you will need to change it also in all the addons).
