General

Language:
Language used in TaskUnifier.
You have to restart TaskUnifier in order to apply the selected language.
Global hot key to add a quick task:
Keyboard shortcut, available even if TaskUnifier is minimized. When pressed, a small window will be displayed where you can enter a quick task.
The keyboard shortcut must be composed of one or two special keys (CTRL, ALT or SHIFT) and another character (A-Z, 0-9).
Minimize to system tray:
If this option is checked, when you minimize TaskUnifier, the software will be minimized in the system tray instead of the system status bar.
Postpone task from current date:
From current date: when you postpone a task, the current date will be used.
From start/due date: when you postpone a task, the start/due date of that task will be used.
If you postpone the start date & due date at the same time this option has no effect, the start/due date of the task will be used.
Show edit window when a task is added:
If checked, the edit task window will appear when you add a task.
Show completed tasks at the end:
Show the completed tasks at the end.
Reset general searchers:
This will delete ALL the searchers list in the category "General". The default searchers will be re-created.
