Manage Task Templates

A template allows you quickly create tasks with pre-defined values.
To manage your templates, go to "File -> Manage task templates".

Title:
The default title for new tasks.
Tags:
The default tags for new tasks.
Separate tags with a comma (,).
Folder:
The default folder for new tasks.
Context:
The default context for new tasks.
Goal:
The default goal for new tasks.
Location:
The default location for new tasks.
Completed:
Defines whether new tasks have to be completed or not.
Due date:
The default due date for new tasks.
The value must be a number of days. The due date will be now + number of days.
Example: if you entered 3 and you create a new task on 01/01/2010, the due date will be 04/01/2010.
Start date:
The default start date for new tasks.
The value must be a number of days. The start date will be now + number of days.
Example: if you entered 3 and you create a new task on 01/01/2010, the start date will be 04/01/2010.
Reminder:
The default reminder (in minutes) for new tasks.
The value must be an integer.
When the due date is less than x minutes after now, TaskUnifier will remind you of this task.
Repeat:
The default repetition for new tasks.
Defines how the task must repeat. Leave this field blank if you don't want the task to repeat.
Repeat from:
Defines how the new tasks will repeat.
- Repeat from due date
- Repeat from completion date
Status:
The default status for new tasks.
Length:
The default length for new tasks.
Priority:
The default priority for new tasks.
Star:
Defines whether new tasks have to be starred or not.
Note:
The default note for new tasks.
