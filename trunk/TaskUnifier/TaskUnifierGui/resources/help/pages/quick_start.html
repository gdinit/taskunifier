Quick Start

TaskUnifier is a task management tool which allows you to manage your tasks.

There are multiple ways to categorize your tasks:

Folders
You can categorize your tasks by folder. A folder can also be considered as a project.
Contexts
A context describes where you can execute a task or what do you need to execute the task.
For example, a context could be "@Work" or "@Home". Contexts could also be "@Developer" or "@Writer".
Locations
Defines the location where the task can be executed.
If you have a task "Buy milk", the location could be "@Grocery".
Goals
Goals help you define what you try to achieve by executing some tasks.
An example of goal could be "Get promoted".


The first thing to do in TaskUnifier is therefore to define these categories.
You can easily manage all your categories in "File -> Manage contexts, folders...".


TaskUnifier offers to possibility to create searchers. A searcher contains two things:
Sorters
Allows you to sort your tasks the way you want.
Filters
Allows you to filter out the tasks you do not want to see.


Searchers allows you to quickly select a set of tasks that matches some conditions.
You can easily add searchers by clicking on the "+" sign at the bottom left of the window.
You can also edit the existing searchers by double clicking on one of them.


Another powerfull feature of TaskUnifier, is to create task templates.
A task template contains pre-defined task values. You can then quickly create new tasks which are pre-filled in with the template values.
Go to "File -> Manage task templates" in order to manage your task templates.


Now your are ready to create your first tasks and notes in TaskUnifier !
